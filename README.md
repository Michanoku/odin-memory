# The Odin Project Memory Card Project - PoKéMeMoRy
A simple memory game using the PokéAPI, implemented with React. 

## Goal
The goal of the game is to click all cards without clicking the same card twice. If the user makes a mistake, the current high score is saved and displayed, and the user can try again with a fresh set of cards. If the user has no more options, they achieve the highest possible score.

## Features
- Cards with Pokémon sprites and background colors matching the Pokémon's types.
- Each Pokémon's cries can be heard when clicking their card. The cries can be muted via a button.
- The game can be restarted by using the refresh button.
- Data received from the API is cached. The same Pokémon are not fetched twice.
